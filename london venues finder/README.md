# London Venues Finder

London Venues Finder is a simple tool concept for finding coworking spaces and flexible venues near London that can host meetups, events, workshops, and hackathons. Users can enter an estimated number of attendees, compare suitable spaces by capacity and location, and open a map view to understand what is nearby before choosing a venue.

## Purpose

The goal is to make venue discovery faster for organisers who need practical spaces with the right size, transport access, and event-friendly facilities. Instead of manually checking many venue pages, organisers can start with capacity and event type, then shortlist locations that fit the experience they want to create.

## Core Features

- Search coworking spaces and flexible event venues across London.
- Filter by estimated attendee capacity.
- Compare venues for meetups, networking events, workshops, and hackathons.
- View nearby transport links, cafes, restaurants, hotels, and useful amenities on a map.
- Store venue details such as neighbourhood, closest station, capacity, website, and event fit.

## Suggested Venue Data

| Field | Description |
| --- | --- |
| Venue name | Name of the coworking space or event venue |
| Area | London area, such as central, east, west, north, or south |
| Neighbourhood | Local area, such as Soho, Shoreditch, Holborn, or King's Cross |
| Closest station | Tube, rail, or Overground station nearby |
| Estimated capacity | Approximate number of attendees the venue can support |
| Event fit | Meetup, hackathon, panel, workshop, launch, or networking |
| Website | Link to the venue or booking page |
| Map link | Link that opens the location in Google Maps |

## Example Capacity Guide

| Event type | Estimated capacity range |
| --- | --- |
| Small meetup | 20-50 attendees |
| Workshop | 30-80 attendees |
| Networking event | 50-120 attendees |
| Hackathon | 50-200 attendees |
| Larger conference-style event | 150+ attendees |

Capacity should be treated as an estimate until confirmed with the venue, because room layouts, catering, equipment, and seating plans can change the real number of attendees a space can support.

## Example Pitch

London Venues Finder helps organisers discover the right coworking spaces and flexible venues for meetups, events, and hackathons. Enter the number of attendees, compare suitable spaces by estimated capacity, and open the map to see what is nearby. It helps teams choose a venue that works for both the event room and the wider attendee experience.
