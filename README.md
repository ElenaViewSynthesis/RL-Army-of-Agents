# RL Army of Agents

**A portfolio of production-grade, multi-agent AI systems for institutional finance, insurance, and autonomous software engineering.**

This monorepo houses a fleet ("army") of specialist LLM agents built around a shared pattern: reasoning-model orchestration, real-time financial data tooling, streaming token delivery, and durable cloud persistence. The flagship system is an **institutional-grade Equity Research agent**, surrounded by domain specialists for **regulatory capital / risk modeling** and **Warranty & Indemnity (W&I) transactional liability**.

---

## Highlights for reviewers

- **Reasoning-model orchestration** — agents run on **NVIDIA Nemotron Ultra 550B** (reasoning + streaming) with a fast **Poolside Laguna** fallback, served through OpenRouter with OpenAI-compatible function calling.
- **Agentic tool-use loop** — the model autonomously decides which financial data to fetch, calls tools in parallel via `Promise.all()`, and reasons over the results before writing.
- **Live market data** — a hardened **Financial Modeling Prep (FMP)** integration: **13 core endpoints** on the free tier (fundamentals, valuation, analyst sentiment, peers, global indices + VIX) plus **9 premium endpoints** (SEC filings, ETF analytics, mutual-fund holdings, ownership-acquisition) for a **22-endpoint** total.
- **Streaming chat specialists** — each domain expert is defined as a durable `.md` system prompt, hot-loaded at startup, and served over **Server-Sent Events** with token-by-token delivery, a live latency timer, and cancellation.
- **Production plumbing** — FastAPI service layer, dual-write persistence to **Supabase** and **AWS S3**, health checks, credit metering, and graceful degradation on premium-tier API limits.

## The fleet

### 🎯 Equity Research Agent — flagship
**`Equity-Research-agent/`** · Node.js + FastAPI · Nemotron / Laguna

Give it a ticker; it returns an institutional-grade research report. The agent fans out across **13 FMP endpoints** in parallel, runs an autonomous agentic loop to decide what else it needs, then synthesizes everything into a **10-section report** with a **BUY / HOLD / SELL** rating and a **12-month price target**.

Report covers: Executive Summary · Company Overview · 4-year Financial Performance · Balance Sheet & Capital Structure · DCF + peer-multiple Valuation · Growth Outlook · Competitive Position · Insider & Analyst Sentiment · Risk Factors · Recent Developments.

```bash
node agent.js NVDA --save --model=nemotron   # reasoning + streaming
node agent.js NVDA --save --model=laguna      # faster, no reasoning tokens
```

### 🛡️ Chief of Regulatory Capital Modelling — risk specialist
**`Equity-Research-agent/agents/chief-capital-modelling-agent.md`**

Handles SCR calibration, EVT tail-fitting, CAT XL reinsurance optimisation, Euler capital allocation, RORAC/hurdle analysis, and systemic-scenario stress testing (e.g. a cloud-provider LLM outage cascading across an insured portfolio) — regulator-ready output for board packs and ORSA submissions.

### 📄 Transactional Liability — W&I Underwriter — streaming chat specialist
**`Equity-Research-agent/agents/transactional-liability-wi-agent.md`**

A senior **Warranty & Indemnity** underwriter for AI-startup M&A: SPA warranty triage, **Rate-on-Line (ROL) pricing**, retention and exclusions structuring, MRC contract-certainty documentation, claims investigation, and portfolio accumulation management. Served as a live streaming chat agent over the FMP-backed API.

## Architecture

```
User / API
   │
   ▼
FastAPI service  ──►  Agent orchestrator (Node.js)
   │                     │  streaming reasoning tokens
   │                     ▼
   │            OpenRouter  →  Nemotron Ultra 550B  /  Laguna
   │                     │  tool_calls (parallel)
   │                     ▼
   │            FMP /stable API — 13 core + 9 premium endpoints
   │                     │  tool results
   │                     ▼
   │            Model reasons → writes report / answer
   ▼
Persistence: Supabase  +  AWS S3   (parallel dual-write, Promise.allSettled)
```

### FMP endpoint coverage — 22 total

**Core (13, FMP free tier):** company profile · real-time quote · income statement · balance sheet · cash flow · key metrics (TTM) · financial ratios (TTM) · DCF valuation · analyst grades · price-target consensus · stock peers · global market indices + VIX · SEC filings (8-K).

**Premium (9, paid FMP plan required)** — the agent passes a `402 Payment Required` through cleanly and degrades to model knowledge rather than crashing:

| Group | Endpoints | Tier |
|-------|-----------|------|
| Ownership & insider | `acquisition-ownership` (5%+ stake filings), insider trades | Premium+ |
| SEC filings | filings by form type (10-K / 10-Q / 13D / 13F / S-1 / DEF 14A), filer-name → CIK search | Premium+ |
| ETF analytics | `etf/holdings`, `etf/sector-weightings`, `etf/asset-exposure`, `etf/info` | Premium+ |
| Mutual funds | `mutual-fund/holdings` | Premium+ |

**Returned metadata — ETF & mutual-fund calls:**

| Endpoint | Returns per record |
|----------|--------------------|
| `etf/holdings` · `mutual-fund/holdings` | asset ticker · name · ISIN · CUSIP · share count · weight % · market value |
| `etf/sector-weightings` | sector · weight % |
| `etf/asset-exposure` (reverse lookup: stock → ETFs holding it) | ETF ticker · share count · weight % · position market value |
| `etf/info` | expense ratio · AUM · issuer · benchmark index · inception date |

## Quick start

```bash
cd Equity-Research-agent
npm install
cp .env.example .env          # add OPENROUTER_API_KEY and FMP_API_KEY

# One-off report from the terminal
set -a && source .env && set +a
node agent.js NVDA --save --model=nemotron

# Or run the streaming chat + research API
bash start.sh                 # → http://localhost:8000/
```
