ETF company URL extraction steps

Source URL:
https://etfdb.com/compare/market-cap/

Goal:
Retrieve URLs from the table in the extracted page content and store them separately
in a list for later Tavily Extract, graph ingestion, or source tracing.

Steps:

1. Extract the page content from the source URL with Tavily Extract.

2. Parse the extracted HTML or page text and locate the ETF company table.
   Prefer structured HTML parsing when available. If Tavily returns cleaned text
   only, identify table rows by repeated company/name/link patterns.

3. For each table row, find the company link.
   Keep only links that point to company or issuer pages. Ignore pagination,
   navigation, ads, social links, and unrelated compare/filter URLs.

4. Normalize every link into an absolute URL.
   If a table link is relative, join it with the ETFDB base URL:
   https://etfdb.com

5. Deduplicate URLs while preserving the original table order.

6. Store the cleaned URLs separately in a list named etf_company_urls.

The following table lists the top 100 largest exchange-traded funds, ranked by assets under management (AUM). The AUM of an ETF is calculated by multiplying shares outstanding by the market price per share. An ETF’s assets will fluctuate based on both changes in the value of the underlying securities and the creation of new shares or redemption of existing shares. It is worth noting that there may be a difference between an ETF’s market capitalization and the net asset value (NAV) of its underlying securities. The difference results in either a discount or a premium in the trading price of the ETF.
